Esta é a página para gerenciar os desafios de limite. Clique em Criar para adicionar um novo desafio. Lembre-se que, para cada limite criado, você deve adicionar as perguntas correspondentes na seção Perguntas.

Aqui você gerencia as perguntas de múltipla escolha que guiam o aluno na resolução de cada limite. Ao criar uma pergunta, use a caixa de busca para associá-la ao Limite correto.

As estratégias são os métodos de resolução (ex: Fatoração, L'Hôpital). Cadastrá-las aqui garante a consistência dos dados em toda a plataforma.

{% elif 'tiposindeterminacao' in request.endpoint %}

Aqui você cadastra os tipos de indeterminação (ex: 0/0) que serão associados a cada limite.
